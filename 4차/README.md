## 4차 과제 수행 내용
- Spring Security를 활용하여, 로그인, 회원가입 기능을 구현
- CommunityUserDetailsService 클래스를 정의하고, UserDetailsService 의 구현체로 선언
- UserRepository 를 통해 받아온 UserEntity 를 User 의 형태로 반환
